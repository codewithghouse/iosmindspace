# Mental Health Assessment Questions and Answers

## Overview
This document contains all the questions and answer options for the 6 mental health assessment categories in the Goodmind platform. Each assessment is designed to evaluate different aspects of mental health and provide appropriate recommendations.

## Contact Information Collection
All assessments end with collecting:
- **Mobile Number** (required)
- **Email Address** (required)

## Assessment Flow
1. User selects assessment type and path ("want help" vs "want to help")
2. Takes questionnaire with questions per assessment
3. Provides contact information
4. System calculates score and determines severity level
5. Results displayed with recommendations to book counseling sessions
6. Results emailed to user
7. Data saved to database

## Scoring System Summary
- **Anxiety**: ≤20 (Mild), 21-32 (Moderate), >32 (Severe)
- **Depression**: ≤15 (Mild), 16-30 (Moderate), >30 (Severe)
- **PTSD**: ≤45 (Mild), 46-65 (Moderate), >65 (Severe)
- **Stress**: ≤40 (Mild), 41-60 (Moderate), >60 (Severe)
- **Relationship**: ≤50 (Mild), 51-60 (Moderate), >60 (Severe)
- **OCD**: ≤25 (Mild), 26-36 (Moderate), >36 (Severe)

## Multi-language Support
The platform supports 4 languages:
- English (en)
- Hindi (hi)
- Telugu (te)
- Urdu (ur)

All questions and answers are localized using the i18n system with translation keys stored in the `locales/` directory.
